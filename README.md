# DonutTab

Chrome extension for new tab Based on ES6, React, Webpack

### Install the local Chrome extension

- Download the package

		cd /somewhere

		git clone git@github.com:zchen9/DonutTab.git
	
- Find Chrome Setting
	
![](images/Chrome_setting.png)
